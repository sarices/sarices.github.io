饭否API应用

sarices
2007-06-15
旧文章

刚看了官方BLOG的介绍，自从饭否开放API以来已经有多种应用了，如Foobar插件，能把正在听的歌发到饭否上，和大家分享你喜欢的音乐，还有Realazy开发的中国人民进行时利用Google地图做的新玩意，能在地图上显示最新的饭否消息，当然你需要在饭否上设定你所在地，我想如果根据IP来得到所在地那就更好了，不过为了安全问题，可能不会公开~还有很多很多有趣的应用，请浏览这个收集饭否API应用的网址。
