利用Https访问维基百科

自从维基百科被封了以后一直都是使用代理访问，已经习惯了使用代理访问了，刚是CNBlog那发现可以利用Https访问维基百科。

Https访问维基百科的地址是：https://secure.wikimedia.org/wikipedia/zh/wiki/

利用这个方法也可以很多网站，如前段时间被封的sourceforge，但前提是网站开了https服务，并打开443端口，因为https使用的是443端口而http使用的是80端口，麻烦的是每打打开一个网页都要确定一次。

Tao.La开发计划

很久没写过为自己而写的程序了，过年的时候和朋友讨论要在网上开一是网店，因为他现在还是上学中，而且一直都在学校做卖精品之类的东东，基本上是做大学城的生意，我们已经连续三年在年宵摊位卖精品的经验，所以就想开一间属于自己的网店，卖些时尚类的东东。这个想法不是一时冲动，已经计划了很久了，连想域名都想了一个多月了，想了很多名字，大多都是中文的名字好听，但注不到.com域名，最后想到有一个国外的域名后缀.la，就注了tao.la这个域名，中文名字：淘啦。

现在定一个大致的方向：

  1. 使用asp+access开发
  2. 不用分类，直接用Tag，生成Tag Cloud
  3. 简洁的界面，简单的注册
  4. 生成HTML或生成XML(asp+XML)/使用cache(待定)
  5. 生成商品Rss，方便订阅(待定)
  6. 加入TrackBack(待定)
  7. XHTML+CSS
    使用asp是为了方便，asp主机到处都是，access数据库可能会改成sql，网站一定要用tao.la访问，如果是用www.tao.la访问直接转向到tao.la，不然就失去了这个域名的意义，暂时就这些，大家有什么意见？可以回复告之!!

Windows Live Messenger Beta 8.0.0566发布

今天早上登录Msn发现提示更新新版本，安装重启后发现界面已经变了，虽然比上一版本清淡了许多但还是觉得上一版本好看，不太喜欢现在的默认的配色把配色改成黑色看好看了许多。
