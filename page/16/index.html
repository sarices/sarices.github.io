我的M6坏了

这段时间很少听MP3了，所以我的M6也只是放在公司的台面上，基本上没动过，昨天得到个消息说新的固件放出来了，对新固件比较期盼的是支持APE，说是mp3，其实我一般听flac或者wav的，基本上很少听MP3格式，因为音质比较差，还是无损的好。下载更新很顺利，不用几分钟就搞好了，因为手头上没有APE的歌，所以用一首wav的转成ape试一下，果然可以播放，没有什么事，没听完一首就狂转APE了，准备下班回家听，谁知道下班回家听的时候才出事，第一着，没听到一关就自动关机了，开始还以为是可能没电了，然后再开机，听第二首，听着听着突然一阵杂音，又关机了，再开，已经开不了了，今天回到公司上网找解决方案，照着做还是不行，看来还是要找魅族了…

越来越看清楚这个世界

最近和合作的公司的某人有点矛盾，当然不只是一点这么简单，矛盾也不是一天两天的事，我不是一个喜欢在老板面前打小报告、落井下石的人，我不出声不代表我会逆来顺受。前几天因为网站新闻的事和她吵了一架，在QQ签名里写下了自己对她的看法，第二天回到公司就收到了对方公司的人的几个留言，都是支持我的，那更坚定了我的想法，有些事情不对就一要说出来，不可能一直受她的威胁，威胁我又怎样？我是不受威胁的，最多就是不做这份工作。能拿我怎样？哭有用吗？

这种人是不值得可怜的，平生最讨厌就是自大而又没能力，没有脑的人，你有能力，有脑自大我还可以接受，但是你一没能力，二没有用脑，脑是拿来用的，不是拿来显摆的，做事要变通一下，不只是在QQ才可以找到人，高材生又怎样，不就是一个小小的本科毕业，考得好成绩不代表有能力，只代表你会考试而已~

这件事不能怪谁，怪公司太复杂了，一直以为林生是一个能看清楚事实的人，不过最后也还是令我失望，只是听了她的话，根本没听我的话，就对另外的老板说，这样对我公平吗？我也是你的员工啊，你让我坐这个位置就是要信任我！你为什么不听听我说呢？现在好了，把所有的交接工作放手出来，我不理了，有什么事不要找我，现在我只有两条路，一条路就是不理，像小薇说的什么事都不关我事，不理那么多，第二条就是辞职算了，这件事再一次打击了我的信心，让我再次看清楚这个世界。

以上言论只是个人见解与公司无关，不要因为我的个人见解影响到两间公司的正常交接。

CSS 属性表 (简表)

属性名称 属性含义 属性值

字体属性（Font）
font-family 使用什么字体 所有的字体
font-style 字体是否斜体 Normal、italic、oblique
font-variant 是否用小体大写 Normal、small-caps
font-weight 字体的粗细 Normal、bold、bolder、lithter等
font-size 字体的大小 Absolute-size、relative-size、length、percentage等

颜色和背景属性
Color 定义前景色 颜色
Background-color 定义背景色 颜色
Background-image 定义背景图案 路径
Background-repeat 重复方式 Repeat-x、repeat-y、no-repeat
Background-attachment 设置滚动 Scroll、Fixed
Background-position 初始位置 Percentage、length、top、left、right、bottom等

文本属性
Word-spacing 单词之间的间距 Normal <length>
Letter-spacing 字母之间的间距 Normal <length>
Text-decoration 文字的装饰样式 None|underline|overline|line-through|blink
Vertical-align 垂直方向的位置 Baseline|sub|super|top|text-top|middle|bottom|text-bottom
Text-transform 文本转换 Capitalize|uppercase|lowercase|none
Text-align 对齐方式 Left|right|center|justify
Text-indent 首行的缩进方式 <length>|<percentage>
Line-height 文本的行高 Normal|<number>|<length>|<percentage>

边距属性
Margin-top 顶端边距 Length|percentage|auto
Margin-right 右侧边距 Length|percentage|auto
Margin-bottom 底端边距 Length|percentage|auto
Margin-left 左侧边距 Length|percentage|auto

填充距属性
Padding-top 顶端填充距 Length|percentage
Padding-right 右侧填充距 Length|percentage
Padding-bottom 底端填充距 Length|percentage
Padding-left 左侧填充距 Length|percentage

边框属性 Border-top-width 顶端边框宽度 Thin|medium|thick|length
Border-right-width 右侧边框宽度 Thin|medium|thick|length
Border-bottom-width 底端边框宽度 Thin|medium|thick|length
Border-left-width 左侧边框宽度 Thin|medium|thick|length
Border-width 一次定义宽度 Thin|medium|thick|length
Border-color 设置边框颜色 Color
Border-style 设置边框样式 None|dotted|dash|solid等
Border-top 一次定义顶端 Border-top-width|color等
Border-right 一次定义右侧 Border-top-width|color等
Border-bottom 一次定义底端 Border-top-width|color等
Border-left 一次定义左侧 Border-top-width|color等
Width 定义宽度属性 Length|percentage|auto
Height 定义高度属性 Length|auto
Float 文字环绕 Left|right|none
Clear 哪一边环绕 Left|right|none|both

分级属性 Display 定义是否显示 Block、inline、list-item、none
White-space 怎样处理空白 Normal、pre、nowrap
List-style-type 加项目编号 Disc、circle、square等
List-style-image 加图案 <url>|none
List-style-position 第二行起始位置 Inside、outside
List-style 一次定义列表 <keyword>|<position>|<url>

《友爱.满分》

http://cachefile2.fs2you.com/zh-cn/download/a1a6d2a543332d8a6b813eb1fe87c065/file.mp3

同事的新歌，支持一下~~

Radio Lady，来自广东电台音乐之声《天生快活人》节目的一对漂亮宝贝——欣欣，活泼爽朗，多年炼就了深厚的主持功力，独特的嗓音令人过耳不忘；林琳，优雅恬静，懂多国语言的美女DJ，纤瘦的身躯里潜藏着巨大的声音爆发力。两位快活的女生一拍即合，在主持、音乐上迸发出夺目的绚烂火花。继《Radio Day》、《水果恋》之后，又一清新甜蜜的单曲《友爱•满分》，在2007这个热辣夏天隆重登场，并成为MUSIC FM“阳光行动2007”主题歌。

Radio Lady以阳光好声音，唤醒您耳朵，激活夏日阳光行动。在《友爱•满分》中，Radio Lady首次尝试对唱，唱功大有长进的两位女生一问一答地讲述了一个关于人生的道理：人生就像天气，既真实而又变幻莫测，幸好在每一次措手不及的时候，总有知心的好友及时预告，无论阴晴雨雪，都有挚爱的好友一路相伴。

新歌再一次体现了欣欣和林琳之间的深厚友谊，也反映了友情在她们心目中的重要地位。她们表示，无论是音乐、主持，还是在人生的道路上，只有两人同心，互相帮助，才能成为“满分”的最佳拍档。

《友爱•满分》的旋律听上去非常熟悉。细听之下可以发现，其实每个星期六下午2点到3点半，扭开收音机调到FM99.3，在由Radio Lady主持的《993斗秀场》节目里，这首歌的高潮部分就一直以“天气预报”歌的形式，出现在每次节目的开场部分。在众多听迷的热情要求下，这首歌经过了多次修改和调整，终于完整地做成了单曲推出。据悉，这首歌还将收录在今年10月2日推出的《天生快活人》节目九周年纪念专辑当中。

作为“阳光行动2007”的主题歌，Radio Lady将在8月的慈善活动中首次现场演绎《友爱•满分》，并把阳光和善心带到贫困山区当中。与此同时，《友爱•满分》即将向广大听迷朋友征集剧场版，唤起大家一同珍惜友情可贵。
